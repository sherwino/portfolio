mccserv.net

This corporate site was built using the popular Content Management System (CMS) Wordpress 4.8.1, and a MySQL 5.6.32 database. The site utilizes HTML5, CSS3, JavaScript ES5, and PHP5.6. Most of the content was uploaded using Filezilla SFTP client to the Apache Linux webserver. Now one of the issues building this company site was related to the fact that it was on a shared host. In order to optimize the load times went ahead and set up Amazon Cloudfront CDN to host all of the static content and take some load off the server. The site is also being cached using WPFC. Small edits to the photos done in Adobe Lightroom, Photoshop, and Gimp.

Project Man | Web App

Project Man

Project Man was designed to solve a simple problem at my old company. The company has been using excel to log and keep track of projects. While Excel can be an extremely powerful tool, it is not designed to be collaborative. You can share files, and you can have multiple people editing at once but there is barely any accountability, and it is difficult to keep track of all of the changes, and who made them. Along with that without constantly protecting the workbook people collaborating can accidentally shift cells and modify the spreadsheet and in ways that can lead to several issues. The web app was designed in Node.js, Express, EJS, HTML, CSS, and JavaScript. It is a simple CRUD application that will let only authorized users make changes, and it will show additional informationt to administrators.
