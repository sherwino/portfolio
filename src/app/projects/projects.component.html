mccserv.net

This corporate site was built using the popular Content Management System (CMS) Wordpress 4.8.1, and a MySQL 5.6.32 database. The site utilizes HTML5, CSS3, JavaScript ES5, and PHP5.6. Most of the content was uploaded using Filezilla SFTP client to the Apache Linux webserver. Now one of the issues building this company site was related to the fact that it was on a shared host. In order to optimize the load times went ahead and set up Amazon Cloudfront CDN to host all of the static content and take some load off the server. The site is also being cached using WPFC. Small edits to the photos done in Adobe Lightroom, Photoshop, and Gimp.
